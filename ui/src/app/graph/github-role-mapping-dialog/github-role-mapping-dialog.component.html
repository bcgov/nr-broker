GitHub Access Help

Users must link their GitHub account to Broker and be assigned a team role that connects them to a repository on the graph to gain access. A user's role within a team is mapped to a corresponding GitHub role. If a user holds multiple roles across the same or different teams, the highest GitHub role is applied.

Repositories must grant access to Broker's GitHub App before enabling sync.
