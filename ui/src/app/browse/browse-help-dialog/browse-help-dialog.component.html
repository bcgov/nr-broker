Browse Help

Overview

The browse table allows you to view and interact with items in a collection. A collection is a group of related items, such as projects, services, environments, or teams.

 Table

The table displays the items in the selected collection. Click anywhere on a row to open the item details.

Collection columns

Each collection has its own set of data fields, some of which are shown as columns in the collection table.

Connections column

Shows the number of upstream and downstream connections for each item:

  • ↑Upstream: connections that target this item
  • ↓Downstream: connections that originate from this item

Action buttons

Each row has buttons to interact with the item:

  • View: opens the item details
  • View on Graph: opens the graph with this item selected
  • Members: available for teams only, shows users assigned to the team
